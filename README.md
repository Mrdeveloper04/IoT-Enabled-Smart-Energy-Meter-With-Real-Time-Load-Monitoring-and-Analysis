# # IoT Enabled Smart Energy Meter With Real-Time Load Monitoring and Analysis

An IoT-based smart energy meter system using ESP8266 NodeMCU and PZEM-004T sensor for real-time monitoring of voltage, current, power, and energy consumption.

## 🚀 Features

- Real-time Voltage Monitoring
- Current Measurement
- Power Consumption Analysis
- Energy Usage Tracking
- IoT Cloud Monitoring using Blynk
- LCD Display Output
- Wi-Fi Enabled ESP8266
- Energy Reset Button
- Remote Monitoring through Mobile App

## 🛠️ Technologies Used

- ESP8266 NodeMCU
- PZEM-004T Sensor
- Arduino IDE
- Blynk IoT Platform
- Embedded C++

## 📦 Components Required

- ESP8266 NodeMCU
- PZEM-004T Sensor
- LCD Display with I2C
- Push Button
- Relay Module
- Jumper Wires
- Breadboard
