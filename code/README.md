# Arduino Code

This folder contains the Arduino source code for the IoT Enabled Smart Energy Meter project.

## Main Features

- Reads Voltage, Current, Power and Energy
- Sends data to Blynk IoT Platform
- Displays values on LCD
- Supports Energy Reset Button

## Required Libraries

Install the following libraries in Arduino IDE:

- ESP8266WiFi
- Blynk
- PZEM004Tv30
- Wire
- LiquidCrystal_I2C

## Board Required

- NodeMCU 1.0 (ESP8266)

## Upload Instructions

1. Open Arduino IDE
2. Install ESP8266 Board Package
3. Install required libraries
4. Select correct COM Port
5. Upload the code to ESP8266
