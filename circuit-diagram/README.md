# Circuit Diagram

This folder contains the circuit diagram of the IoT Enabled Smart Energy Meter.

## Connections

### PZEM-004T ↔ ESP8266

| PZEM | ESP8266 |
|---|---|
| VCC | 5V |
| GND | GND |
| TX | D6 |
| RX | D7 |

### LCD ↔ ESP8266

| LCD | ESP8266 |
|---|---|
| SDA | D2 |
| SCL | D1 |

### Push Button

| Button | ESP8266 |
|---|---|
| One Terminal | D3 |
| Other Terminal | GND |
